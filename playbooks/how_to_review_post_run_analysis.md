# Playbook: Review Post-Run Analysis

## Use when
Reviewing failure cohorts, performance reports, or remediation proposals after a run.

## Load
Run manifest/summary, deterministic cohort definitions, representative redacted evidence, analysis threads, evaluation metrics, and remediation schema.

## Procedure
1. Verify counts and deterministic metrics independently of model prose.
2. Check cohort membership and representative selection.
3. Separate observations, calculations, hypotheses, unknowns, and recommendations.
4. Assess whether the cause is input, prompt, model, validator, orchestration, or infeasible goal.
5. Require a regression fixture and safe sample scope for any proposed change.
6. Accept, revise, defer, or reject the proposal; analysis cannot execute it.

## Output
A reviewed advisory disposition with evidence, risks, required approvals, and next action.
